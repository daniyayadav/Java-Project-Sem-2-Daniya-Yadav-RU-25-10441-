# Digital Stopwatch Logic Simulation (Java OOP)

## 📌 Description
This project implements a Digital Stopwatch using Object-Oriented Programming in Java. It simulates real-world stopwatch behavior using system time.

## ⚙️ Features
- Start the stopwatch
- Stop (pause) the stopwatch
- Resume timing
- Reset stopwatch
- Get elapsed time in milliseconds

## 🧠 Concepts Used
- Encapsulation
- State Management using boolean flag
- Time tracking using System.currentTimeMillis()
- Object-Oriented Design

## ▶️ How to Run

1. Compile:
   javac StopwatchApp.java

2. Run:
   java StopwatchApp
